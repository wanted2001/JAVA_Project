배열 관련 메서드 : 유사배열에서는 사용제한이 되는 메서드도 있음.

  • array.push(value) : 배열의 마지막에 value 값을 추가 => 배열의 길이를 리턴
  • array.pop() : 배열의 마지막 요소를 제거 => 배열의 마지막 요소를 리턴
  • array.shift : 배열의 첫요소를 제거 => 배열의 첫 요소를 리턴 (pop()과 반대)
  • array.unshift(value) : 배열의 첫요소에 값을 추가 => 배열의 길이 리턴 (push()와 반대)
  • join(value) : 배열 사이의 value를 추가하여 문자열로 리턴
