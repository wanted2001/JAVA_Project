• array.reduce(callback function(total, value, index, self),[init value]); 연산의 누적값을 리턴
  • array.reduceRight() : 배열의 우측에서 좌측으로 대입
  • array.every(callback function(value, index, self)) 콜백 함수의 리턴 조건에 모든 배열의 값이 만족하면 true / false를 리턴
  • array.some(callback function(value, index, self)) 콜백 함수의 리턴 조건에 하나라도 값이 만족하면 true / false를 리턴
  • array.indexOf('str',[start index]) 해당 문자열이 존재하는 index를 리턴
  • array.lastindexOf('str',[start index]) 뒤에서 부터 문자열 탐색 index를 리턴
  • array.find(callback function(value, index, self)) 검색 조건에 만족하는 첫번쨰 값을 리턴
  • array.findIndex(callback function(value, index, self)) 검색 조건에 만족하는 첫번째 번지를 리턴
  • array.form('객체') : 순서가 있거나 순환 구조를 갖고 있는 객체를 배열로 반환
  • array.keys() : 배열이 사용중인 key(index)를 검색하여 새로운 배열로 리턴
  • array.entries() : index : value의 구조값을 만들어 새로운 배열에 담아 리턴
  • array.include("str") : 배열의 str 값이 있으면 true / false
