String : 가장 많이 다루는 데이터 타입

  • java에서의 String method와 거의 유사함.
  • 문자열 데이터는 index (0부터 시작)를 가지고 있음.
  • 문자열 값 내에서 (br)를 하는 것을 추천하지 않음.
  • new string() : 객체 타입으로 선언가능 / 추천하지않음
  • length : 문자열의 총길이
    • 추출 관련 메서드 : substring(), substr(), slice()
    • 문자를 대체하는 replace(old,new) : 처음 찾는 값만 가능
    • trim() : 양쪽 공백 제거
    • padstart(길이, 채울문자) : / padEnd(길이 채울문자) 010101 - 4******* / ****시
    • CharAt(index) / indexOf()
    • startwith("문자열") / endwith("문자열") 해당 문자열 앞 / 뒤에 문자열이 존재하는지 확인하는 메서드 return true / false
