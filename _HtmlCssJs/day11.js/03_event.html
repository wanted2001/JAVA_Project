이벤트 객체는 전역객체로써 모든 태그에 객체를 참조 시킬 수 있다.

  • a tag / button tag에 이벤트를 주로 참조시킴 클릭에 대한 반응을 할 수 있게 함.
  • a tag는 href 속성에 의해 자체이동(동기화 방식)이 구현 되어있어서 별도의 이동을 막는 코드가 필요
  • 객체에 이벤트를 부여하는 방식이 3가지
  • 각자 목적과 작동방식이 다름
  • 화면의 영역을 Document영역(객체) 라고 한다.
  • DOM 영역
  • tag는 html element, 속성은 attribute, 일반 텍스트는 textNode 라고 함.
  • html로 출력하기 : innerHTML라는 property를 사용
  • text로 출력하기 : innerText property를 사용
  • html element 를 가져올떄 id로 가져오는 방식 getElementByid('idName')
  • html element 를 가져올떄 class로 가져오는 방식 querySelector('selector') => #(id), .(class) 선택자 가능
