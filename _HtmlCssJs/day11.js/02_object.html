객체(Object)

  • object : 데이터의 구조 상세 특징으로 표현하려는 형식
  • const으로 선언하는 것이 일반적 : 객체의 구조를 지속성있게 유지하고, 확보하기 위함.
  • {속성명 : 값, 속성명 : 값, 속성명 : 값, ...}
  • 속성명 : key 값 : value
  • 같은 객체안에같은 속성명은 불가는 (key는 중복 불가능)
  • 객체 내부에서 속성 값으로 사용할 수 있는 데이터의 제한이 없음.
  • 객체의 속성값에 함수를 쓸수 있음 익명함수를 정의 => 메서드라고 부름 메서드의 이름은 속성명이 됨.
