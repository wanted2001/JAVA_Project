form 태그를 통해 데이터 받아오기

  • input, textarea, select, html의 element 속성, 텍스트 데이터를 받아오기
  • input tag는 value 속성으로 값이 맵핑이 되어 있어서 value로 값을 가져올수 있음.
  • select=> option / radio/ checkbox value 값을 별도로 지정해야함.
  • value / innertext 값은 모두 String으로 처리됨.
  • string 처리된 value 값을 숫자로 사용하기 위해서는 변환을 해줘야함.
