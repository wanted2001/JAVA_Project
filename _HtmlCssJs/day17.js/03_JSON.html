JSON (JavaScript Object Notation) : 자바 스크립트의 객체 표기법

  • 데이터의 요청및 응답처리에 필요한 데이터의 표준 형식을 제공하는 타입
  • 백엔드(서버)언어 종류에 상관없이 데이터를 송수신 단말 형식으로 통일
  • 백엔드(서버)에서는 무조건 String 처리 - 화면(JavaScript)은 Object로 처리
  • 서버로 데이터를 보낼때 > object -> string > JSON.stringify(jsObject);
  • 서버로 부터 데이터를 받을때 > string -> object > JSON.parse(stringdata);
