자바 스크립트의 파라미터 특징 / 기능

  • 파라미터의 개수와 관계없이 함수 이름이 같다면 호출가능
  • 파라미터의 개수가 불일치하여 받지 못하는 값이 있다면 undifined 처리
  • 값을 받아오지 못하는 파라미터가 undifined 처리를 막기 위해 기본값 설정 가능. => optional parameter
  • 모든 파라미터는 aruments라는 프로퍼티를 이용하여 파라미터를 관리 arguments라는 배열로 담겨서 관리
  • optional parameter는 저장안됨.
  • optional parameter는 다른 파라미터보다 가장 뒤에 있어야함.
  • rest parameter : 남는(이외의, 남는것을) 파라미터
  • 사용할 파라미터는 일반 파라미터로 배정하고 나머지 파라미터들을 통합하여 하나의 이름으로 배정하는 기법
  • ... : spread 연산자라고 하여 배열 객체를 복제하는데도 사용함.
