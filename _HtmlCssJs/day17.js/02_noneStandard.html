비표준 속성(non-standard)

  • html 태그의 속성이 제한적이므로 데이터를 관리 차원에서 개발자들이 직접 속성을 만들어 사용할 수 있는 방법을 제공
  • 너무 과도한 사용은 가독성과 개발효율이 떨어짐 사용은 적당히,
  • 태그 속성의 이름을 data-변수명 = "${값}" -(하이픈) 뒤에 변수명을 사용 value의 값을 인식할 수 있도록 설계되어 있음.
  • 값을 읽어올 경우 : dataset.변수명 값이 리턴됨. dataset property를 사용하여 읽어들임
  • 변수명에 대문자는 허용안함. : 대문자를 사용해도 소문자로 변환하여 읽어 올수 있음.
