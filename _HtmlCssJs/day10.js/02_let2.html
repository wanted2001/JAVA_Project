자바스크립트의 변수 사용

  • 값을 담고있는 메모리 공간의 이름
  • JAVA => 자료형을 작성후 뒤에 변수명 작성 SCRIPT => 변수를 선언하는 역할만 함 (자료형 X)
  • var, let(ES6 var는 권장하지 않음), const
  • let 자바에서 사용되는 지역변수의 개념
  • var name = 'hong'; var name = 'kim'; => 가능
  • let 중복선언 불가 재할당 가능 let name = 'hong'; let name; X name = 'kim'; O
  • const 재선언 불가 중복선언 불가 재할당 불가 const name = 'hong'; name = 'kim'; X
  • 스코프 scope:사용범위 ( )안에서만 가능 스코프를 벗어나면 재선언 가능
  • const : 고정된 값을 사용하는 규칙, 참조, 객체, 엘리먼트 정의시 사용
