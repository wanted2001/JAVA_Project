window 메서드 종류

  • window.open('url','window name', 'option(size, scroll...)');
  • window.close(); : 창 닫기 잘 쓰진 않음.
  • alert(); / prompt(); input포함 / confirm() 확인/취소 창
  • setInterval(function, millisecond); 일정 간격으로 지속적으로 실행문을 실행시킬때 사용
  • setTimeout(function, millisecond); 일정시간동안 실행문을 실행시킬떄 사용
